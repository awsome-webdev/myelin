Quizlet in the past has been a great option for studying and learning material before a quiz or for just general learning, they dominated with their different learning modes and content that anyone could make and study. But now their main focus seems to be how many ads they can shove in your face and what you can pay for. Not to mention their website so full of adware and other trackers it takes 30 minutes to load on a average school Chromebook.
So I built this to be a better replacement for Quizlet and has features such as:

- Block blast mode for studying and playing at the same time
- test mode with text embedding model for better test options
- memory score based on test scores and frequency 
- Quizlet import via PDF 
- AI agentic set creation 
- Global set sharing mechanism
- A full offline mode and the ability to save sets offline
- request caching so no information is lost

## Install

Run this command to clone and then start the install process

```

git clone "https://github.com/awsome-webdev/myelin"

cd myelin

sudo bash install.sh

```

after you run the install script you should be prompted for keys for Hackclub AI and Hackclub search and a port to run on

this will also make a systemctl service
